# CP3402 Supplementary Assignment Report

## Introduction

This project involved the development of a WordPress website using a modern web development workflow. The project was completed using Docker, GitHub, Astra Theme, and a custom child theme.

The goal of the assignment was to demonstrate practical web development, deployment preparation, project management, and version control skills.

---

## Local Development Environment

A local WordPress development environment was created using Docker Desktop and Docker Compose.

The environment included:

* WordPress container
* MySQL database container
* Localhost deployment

Using Docker improved:

* portability
* consistency
* development workflow
* environment management

---

## WordPress Theme Development

The Astra parent theme was installed and customized using a custom Astra child theme.

The child theme included:

* custom styling
* typography improvements
* navigation customization
* footer customization
* responsive design improvements

Using a child theme follows professional WordPress development standards because changes remain separate from the parent theme.

---

## Website Design

The website was designed to be:

* modern
* responsive
* user-friendly
* goal-driven

The homepage includes:

* hero section
* navigation menu
* service information
* footer content
* community-focused branding

The design focused on readability, accessibility, and user engagement.

Git and GitHub were used throughout the project to manage version control and workflow management.
