# CP3402 Supplementary Assignment

## Project Overview

This project is a WordPress website developed for the CP3402 Supplementary Assignment. The website was created using Docker, WordPress, Astra Theme, and a custom child theme.

The project demonstrates:

* Local development workflow
* WordPress customization
* Child theme development
* GitHub version control
* Project management workflow
* Deployment preparation

---

## Technologies Used

* Docker Desktop
* WordPress
* Astra Theme
* Astra Child Theme
* GitHub
* GitHub Projects
* Visual Studio Code

---

## Local Development Setup

### Run Docker Containers

```bash
docker-compose up -d
```

### Open Local Website

```text
http://localhost:8080
```
